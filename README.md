### Getting Started  
> Build & Serve `store` on your system  

```sh
git clone https://github.com/wayneseymour/store.git
cd store
npm install
bower install
grunt
npm start
``` 

### Now browse to http://127.0.0.1:3030 and click on the `store` button on the top right.
